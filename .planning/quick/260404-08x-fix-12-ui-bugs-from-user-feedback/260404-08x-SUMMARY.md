# Quick Task 260404-08x: Fix 12 UI Bugs from User Feedback Summary

Fixed 12 CSS/JS bugs across form, feedback, and display components based on real user testing feedback, polishing components to production quality.

## Task Results

### Task 1: Fix form component bugs (toggle, button, slider, dropdown)
**Commit:** 1c5fa74

- **Bug 1 - Toggle animation:** Overrode `attributeChangedCallback` to do lightweight DOM updates (toggle classes, swap icon opacity) instead of full re-render via `_doRender()`, so CSS transition on `.td-toggle-thumb` plays smoothly
- **Bug 2 - Toggle custom color on track:** Added `color` attribute handling in `attributeChangedCallback` to call `_injectStyle()` without full re-render
- **Bug 3 - Button disabled cursor:** Removed `pointer-events-none` from disabled state classes; the native `disabled` attribute already prevents clicks, and `pointer-events-none` was blocking `cursor: not-allowed` from showing
- **Bug 4 - Slider hover effect:** Removed mouseenter/mouseleave event listeners that applied `scale(1.15)` transform on hover, causing thumb to jump
- **Bug 5 - Dropdown bouncing:** Added `transition: none;` to menu element's inline styles to prevent CSS transitions from causing rubber-band effect during scroll repositioning

### Task 2: Fix feedback component bugs (loading, toast, tooltip, modal)
**Commit:** 4517302

- **Bug 6 - Loading spinner:** Moved `<style>` block with keyframes from inside overlay element to `<head>` so animations are always parsed; fixed `hidden flex` class conflict by setting `display` explicitly on show/hide
- **Bug 7 - Toast spam lag:** Added staggered queue mechanism: `show()` pushes to `_pendingQueue`, flushes via `setTimeout` with 80ms stagger between DOM operations; prevents 8 simultaneous DOM operations + RAF callbacks
- **Bug 8 - Tooltip arrow:** Replaced rotated 8x8 diamond (`transform: rotate(45deg)`) with CSS border triangle (`width:0; height:0; border: 6px solid transparent`); arrow direction set dynamically in `position()` via border-color; eliminates diamond artifact entirely
- **Bug 9 - Remove TdModal.loading:** Deleted `static loading()` method and `_closeAllLoadingModals()` from TdModal; updated modal stories to use `TdLoading.show()/hide()`; updated demo.html to use TdLoading

### Task 3: Fix display component bugs (table diacritics, tabs animation, empty state icon)
**Commit:** 49d1441

- **Bug 10 - Table "muc" diacritics:** Changed default item-label from `'muc'` to `'mục'` in both `td-pagination.js` (default getter) and `td-table.js` (`_renderPagination`)
- **Bug 11 - Tabs sliding indicator:** Added `.td-tabs-indicator` div with absolute positioning, CSS transition for left/width; `_updateIndicator()` method calculates active button offsetLeft/offsetWidth; called in `afterRender()` (via rAF) and `_switchTab()` for smooth sliding animation
- **Bug 12 - Empty state icon:** Replaced inbox-with-arrow SVG (viewBox 0 0 24 24) with open-box-empty SVG (viewBox 0 0 64 64) featuring a box with shelf divider and faint upload arrow at 0.4 opacity
