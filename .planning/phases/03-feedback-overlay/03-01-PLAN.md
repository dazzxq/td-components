    Create `src/feedback/td-modal-stack.js` — port from DCMS `ModalStackManager` to `TdModalStackManager`.

    This is a standalone utility class with static methods (does NOT extend TdBaseElement per D-03 architecture note).

    Port these features exactly from DCMS source:
    1. **Static state:** `stack = []`, `BASE_Z_INDEX = 9999`, `Z_INDEX_INCREMENT = 100`, `BACKDROP_BASE_OPACITY = 0.5`, `BACKDROP_OPACITY_INCREMENT = 0.05`
    2. **generateId()** — unique ID via `Date.now() + Math.random().toString(36)`
    3. **push(modalInstance)** — calculate z-index = BASE + (stackSize * INCREMENT), backdrop opacity = min(0.5 + stackSize * 0.05, 0.8), apply to element, lock body scroll on first modal
    4. **pop()** — remove top, unlock scroll if stack empty
    5. **getTop()** — return top of stack or null
    6. **getStackSize()** — return stack.length
    7. **removeById(modalId)** — splice from stack, recalculate z-index and backdrop opacity for remaining modals, unlock scroll if empty
    8. **closeAll()** — iterate stack top-to-bottom, pop each, run exit animation (scale+opacity), remove DOM after 200ms, call onClose callbacks, reset body overflow
    9. **ensureScrollState()** — safety check for scroll lock desync

    Rename class to `TdModalStackManager`. Rename ID prefix to `td-modal-`. Export as named export.

    Do NOT port `init()` method (it was a no-op in DCMS for backward compat).
  </action>
