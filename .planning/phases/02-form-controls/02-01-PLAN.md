      - backdropFilter: `blur(8px)`
      - boxShadow: layered shadows with inset highlight
      - hoverBoxShadow: elevated shadows on hover
    - Glass style map (port EXACTLY from DCMS glassStyles object):
      - primary: bg `rgba(31,41,55,0.85)`, text-white
      - secondary: bg `rgba(0,0,0,0.04)`, text-gray-700
      - success: bg `rgba(34,197,94,0.85)`, text-white
      - danger: bg `rgba(239,68,68,0.85)`, text-white
      - info: bg `rgba(59,130,246,0.85)`, text-white
      - warning: bg `rgba(249,115,22,0.85)`, text-white
    - Custom color: when `color` attribute set, use it as backgroundColor, auto-calculate text contrast via luminance formula
    - Size classes: sm = `px-3 py-1.5 text-xs`, md = `px-5 py-2.5 text-sm`, lg = `px-6 py-3 text-base`

    - Loading state: `loading` boolean attribute shows spinner SVG + "Dang xu ly..." text, disables button
    - Disabled state: `opacity-50 cursor-not-allowed pointer-events-none`
    - Full width: `full-width` boolean attribute adds `w-full`
    - Hover/focus effects: apply via inline styles in afterRender using `this.listen()` for mouseenter/mouseleave/focus/blur
    - Click handler: emit `click` event via `this.emit('click', {})` — per D-02, D-03
    - Include static helper `_getContrastColor(color)` and `_hexToRgb(hex)` ported from DCMS
    - Include `setLoading(bool)` and `setDisabled(bool)` methods on the element

    IMPORTANT: Glass styles must be applied via inline `style` properties in render/afterRender since they use rgba values not available as Tailwind classes.
